# Chinese Natural Writing（中文自然写作）

> 撰写没有"AI 味"的中文文章——让 AI 写的东西读起来像真人写的。

## 概述

这个 skill 解决的是大语言模型生成中文内容时普遍存在的"机械感"问题：语法完美但毫无个性，立场周全但缺乏灵魂，结构工整但读起来像公文模板。它提供了一套系统性的写作原则和自查方法，帮助 AI 输出更自然、更有"人味"的中文。

**触发词**：写一篇、帮我写、改得自然一点、去掉 AI 味、像真人写的、humanize、太 AI 了

## 核心原理

所谓"AI 味"，本质是概率模型在追求最安全、最高平滑度输出时留下的机械印记。这个 skill 从语言学的根源出发，分析了 AI 中文写作的七大病灶及其成因：

| 病灶 | 根因 |
|------|------|
| 过度使用"然而""此外""总而言之" | AI 缺乏宏观语境理解，靠显性连接词维持表面连贯 |
| 大量翻译腔动词（击穿、赋能、拆解） | 底层英文语料的直译映射 |
| 反复使用"不是……而是……"等强反转句式 | 模板化追求"深度感" |
| 句长高度均一 | 缺乏人类写作的自然节奏波动 |
| "端水式"中立 | 安全对齐策略导致拒绝表态 |
| 宏观概念空转 | 缺少具体事例锚定 |
| 标点符号泛滥 | 用破折号、引号替代自然语言过渡 |

## 七大写作原则

1. **逻辑连接**：删掉至少一半的段落起手连接词，让内容语义自然承转
2. **用词**：说人话，拒绝翻译腔动词和抽象名词堆叠
3. **深度来自洞察**：不依赖"不是……而是……"等模板句式
4. **句长与节奏**：长短交错，制造阅读的"呼吸感"
5. **立场**：敢于表态，允许写得"不全面"
6. **细节**：每个核心观点挂靠具体的、可感知的事例
7. **标点**：求精不求多，用加粗代替引号

## 适用场景

- 撰写中文博客、公众号文章、知乎回答
- 改写 AI 生成的初稿，去机械化风格
- 写中文论文、报告等需要自然表达的场景
- 任何需要让 AI 中文输出"更像人写的"场景

## 优点

- **系统性强**：不是零散的技巧堆砌，而是从语言学根源分析问题
- **可操作**：每条原则都有具体的"怎么做"和自查标准
- **有优先级**：按重要性排列，不会让人无从下手
- **理解驱动**：每条都解释"为什么"，帮助使用者内化原则而非机械执行

## 可能的问题

- 原则较多（七条），初次使用者可能记不住，建议先从前三条开始实践
- "说人话"标准主观性强，不同读者对"自然"的定义不同
- 在正式学术写作场景中，部分原则（如减少连接词、敢于表态）需要适度调整
- 对于非中文母语使用者，判断"翻译腔"可能有难度
